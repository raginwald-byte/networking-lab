# Common Network Ports

## What is a well-known port?

Many Internet services use standard port numbers.

These ports allow computers to know which service they are trying to communicate with.

Some of the most common ports are:

| Port | Service |
|------:|---------|
| 21 | FTP (File Transfer Protocol) |
| 22 | SSH (Secure Shell) |
| 53 | DNS (Domain Name System) |
| 80 | HTTP (HyperText Transfer Protocol) |
| 443 | HTTPS (HyperText Transfer Protocol Secure) |

## Why are these ports important?

When a client connects to a server, it must specify the correct port.

For example:

- A web server using **HTTP** listens on **port 80**.
- A secure website using **HTTPS** listens on **port 443**.
- A DNS server listens on **port 53**.

Using the correct port allows the requested service to receive the incoming data.

## Summary

An IP address identifies a device.

A port identifies the service running on that device.

Together, they allow computers to communicate with the correct application.
