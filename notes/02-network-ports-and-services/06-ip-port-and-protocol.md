# IP Address, Port and Protocol

## Three essential elements of a network communication

A network communication is not defined only by an IP address.

Three elements are required:

- **IP address** → Identifies the device.
- **Port** → Identifies the application or service.
- **Protocol** → Defines how data is transmitted.

These three elements work together to establish communication between two devices.

## Example

When accessing a secure website:

- IP address → Identifies the destination server.
- Port **443** → Identifies the HTTPS service.
- TCP → Ensures reliable data transmission.

Each element has a specific role and is necessary for successful communication.

## Summary

To understand a network connection, it is important to identify:

- The device (**IP address**)
- The service (**Port**)
- The communication method (**Protocol**)
