# What is a Network?

## Definition

A computer network is a group of devices that can communicate with each other.

Examples of devices connected to a network include:

- Computers
- Smartphones
- Tablets
- Smart TVs
- Game consoles
- Printers

## Local Area Network (LAN)

A **LAN (Local Area Network)** is a network that covers a limited area.

Examples:

- A home network
- A company network
- A school network

Devices inside the same LAN can exchange data directly.

## The Internet

The Internet is a worldwide network that connects millions of local networks (LANs) together.

It allows devices from different locations to communicate with each other.
