# Default Gateway

## Definition

The default gateway is the exit point from a Local Area Network (LAN) to other networks, including the Internet.

In most home networks, the default gateway is the router.

Example:

192.168.1.1

Without a default gateway, a computer can communicate only with devices inside its local network.
