# DNS (Domain Name System)

## What is DNS?

DNS stands for **Domain Name System**.

It acts like the Internet's phone book.

Instead of remembering IP addresses, users type domain names such as:

- google.com
- github.com
- wikipedia.org

The DNS server translates the domain name into its corresponding IP address.

Example:

google.com → DNS → 142.xxx.xxx.xxx

This process is called **name resolution**.
