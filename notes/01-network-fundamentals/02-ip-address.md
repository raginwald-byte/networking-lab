# IP Address

## Definition

Every device connected to a network has an IP address.

An IP address uniquely identifies a device on a network.

Example:

192.168.1.25

## Why is an IP address necessary?

Without an IP address:

- A device cannot send data.
- Other devices cannot send data back.
- The device cannot be identified on the network.

Example:

- Router: 192.168.1.1
- Phone: 192.168.1.10
- Computer: 192.168.1.25

Each device has its own unique address.
