# IP Addresses and Communication

## Original handwritten diagram

This diagram illustrates how different devices communicate using IP addresses.

### What this diagram shows

- Every device connected to a network has an IP address.
- Devices exchange data by sending packets to one another.
- Websites and online services are also reached through IP addresses, even if users access them through domain names.
