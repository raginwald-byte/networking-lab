# Network Fundamentals

## Original handwritten diagram

This diagram was drawn by hand while learning the fundamentals of computer networking.

### What this diagram illustrates

- A LAN (Local Area Network) is a group of connected devices.
- The Internet connects many LANs together.
- Data leaves a local network through its gateway before reaching other networks.
