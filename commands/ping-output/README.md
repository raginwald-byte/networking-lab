# ping

## Description

This example shows the execution of the `ping` command against `google.com`.

The command checks whether the destination host is reachable and measures the round-trip time of the packets.

## Command

```cmd
ping google.com
```

## What this example demonstrates

- DNS resolution of `google.com`
- Successful communication with a remote host
- Round-trip time (latency)
- Packet transmission statistics
- No packet loss

## Why use `ping`?

The `ping` command is commonly used to:

- Verify network connectivity
- Test whether a host is reachable
- Measure network latency
- Troubleshoot connectivity issues
