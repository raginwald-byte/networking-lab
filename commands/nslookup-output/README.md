# nslookup

## Description

This example shows the execution of the `nslookup` command on Windows using `google.com` as the target.

The command queries a DNS server to retrieve the IP addresses associated with the domain name.

## Command

```cmd
nslookup google.com
```

## What this example demonstrates

- The DNS server used to resolve the query
- The IPv4 and IPv6 addresses associated with `google.com`
- How a domain name is translated into IP addresses

## Why use `nslookup`?

The `nslookup` command is commonly used to:

- Verify DNS resolution
- Retrieve the IP addresses of a domain
- Troubleshoot DNS-related issues
- Check which DNS server is answering the request
